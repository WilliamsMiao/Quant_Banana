# Quant_Banana 🍌

一个专注于量化交易策略开发和回测的Python框架。

## 项目结构

```
Quant_Banana/
├── src/                    # 源代码目录
│   ├── data/              # 数据处理模块
│   ├── strategies/        # 交易策略
│   ├── backtesting/       # 回测框架
│   └── utils/             # 工具函数
├── docs/                  # 文档
├── tests/                 # 测试文件
├── examples/              # 示例代码
├── requirements.txt       # 依赖包
└── README.md             # 项目说明
```

## 快速开始

1. 克隆项目
```bash
git clone git@github.com:WilliamsMiao/Quant_Banana.git
cd Quant_Banana
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行示例
```bash
python examples/basic_strategy.py
```

## 功能特性

- 📊 多种数据源支持（Yahoo Finance, Tushare, AKShare等）
- 🔧 丰富的技术指标库
- 📈 完整的回测框架
- 🤖 机器学习策略支持
- 📝 详细的文档和示例
